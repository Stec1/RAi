# RAi v1 documentation — frozen archive

This directory holds the complete documentation of RAi v1: the original AI-research MVP plan and
the nine pivot patches (PP-01…PP-09) that turned it into the terminal-and-sphere universe. It is
frozen history — nothing here is updated, and nothing here describes the current product.

RAi was reset at GENESIS (see `../../decision-log.md`, R-DL-01). The v1 decision log
(`decision-log.md` in this directory, DL-01…DL-50) remains a historical record and is superseded
by the R-series log. The canonical v2 documents live in `docs/` at the repo root of this tree.
